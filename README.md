# Pokédex App

A modern mobile Pokédex application built with React Native and Expo.

This app allows users to explore Pokémon information through an interactive and visually engaging interface, including Pokémon details, type filtering, favorites, daily featured Pokémon, and more.

---

## Features

### Home Page

- Daily **Pokémon of the Day**
- Recently viewed Pokémon history
- Browse Pokémon by type
- Fun Pokémon facts card

### Pokédex

- Complete Pokémon list
- Search Pokémon by name
- Filter Pokémon by type
- Responsive Pokémon grid

### Pokémon Details

Each Pokémon has its own details page containing:

- Height, weight and ability
- Base stats
- Evolution chain
- Moves list
- About section
- Favorite button

### Favorites

- Save Pokémon to your favorites list
- Persistent local storage
